# Bio links и bit.ly — сравнение

Краткое сравнение **Bio links** (этот проект) и **bit.ly** для пересылки коллегам или партнёрам.

---

## Общее (похожие возможности)

| Возможность | Bio links | bit.ly |
|-------------|-----------|--------|
| Короткая ссылка → редирект на целевой URL | да (`/r/{slug}`) | да (`bit.ly/...`) |
| Подсчёт кликов | да | да |
| API для автоматизации | да (JSON API v1 + токен) | да (развитый API) |
| Экспорт данных | CSV (ссылки, клики, сводка) | да (зависит от тарифа) |

---

## Чем Bio links отличается в плюс (заточка под нашу задачу)

| Особенность | Bio links | bit.ly |
|-------------|-----------|--------|
| Много ссылок на одну цель из разных аккаунтов соцсетей | массовое создание + поле «аккаунт» | через API/импорт, без такой специализации |
| Автоопределение платформы (TikTok, Instagram, YouTube и т.д.) | да | нет |
| Группировка ссылок по **профилям** | да | workspaces/кампании — другая модель |
| Свой сервер и своя БД (Railway, приватный доступ) | да | SaaS, данные у bit.ly |
| Единая логика с внутренним dashboard (account stats) | да | нет |

---

## Чего нет по сравнению с bit.ly

| bit.ly умеет | Bio links (сейчас) |
|--------------|-------------------|
| Свой / брендовый домен (`go.brand.com`) | только домен деплоя (например Railway) |
| Выбор короткого имени ссылки (`/my-promo`) | случайный `slug` |
| QR-коды | нет |
| UTM-конструктор и маркетинговые кампании | нет |
| Страница link-in-bio | нет |
| Команды, роли, SSO, корпоративный биллинг | один админ + API-токен |
| A/B-тесты, ротация ссылок | нет |
| Срок действия ссылки, пароль на ссылку | нет |
| Deep links, retargeting, мобильная маршрутизация | нет |
| Глобальный CDN и «узнаваемый» домен bit.ly | зависит от хостинга |
| Продвинутая аналитика и интеграции (GA, Meta и т.д.) | базовая статистика + CSV |
| Публичный SaaS с тарифами и поддержкой | self-hosted / приватный инструмент |

---

## Итог в двух предложениях

**bit.ly** — универсальный публичный сервис сокращения ссылок для маркетинга: брендовые домены, QR, команды, расширенная аналитика.

**Bio links** — внутренний инструмент под поток коротких ссылок из соцсетей на одну цель: профили, платформы, массовое создание, своя статистика и API без подписки на bit.ly.
